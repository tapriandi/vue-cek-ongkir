# vue-cek-ongkir

### Ingredients
Fontend
```
- Vue Js
- TailwindCss
- Vue Select
```
Backend
```
- Express Js
```

### Project setup
```
/server  yarn install

/client  yarn install
```

### Compiles and hot-reloads for development
```
/server  node app.js

/client  yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
